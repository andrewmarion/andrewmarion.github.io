Cholera Outbreak: London, 1854



Walk Through

Background

Dr. John Snow created an incredible visual to help find the cause of the Chlorea outbreak in London in 1854. He showed the importance of a good visualization to uncover stories data is trying to tell.

Now, in 2022, technology has advanced immensely. We can now create better visuals with interactive elements to better tell the story the data is telling and find new findings.

This is why I created a visual using D3 to help find those harder-to-find stories through a creative, interactive visual.

I began thinking of the important aspects of the data. We have age, gender, death date, and location of the death. These can be compared to the location of the pumps. In John Snowâ€™s visual, he showed the deaths in relation to the pumps. He did not have any way to signify age, gender, or death date. Therefore, I can use those variables to tell more of a story.

Exploratory Questions

  • Does age matter?
  • Does gender matter?
  • Is there a spike in deaths like a typical outbreak?
  • How does the distribution of deaths change with the date?

    • Design

    Design #1

    First layout: The first layout would be a visual where everything was interactive. All visuals would change depending on where the mouse hovered. This seemed to be more complicated to build and more complicated to understand as a user. Therefore, I moved on to the second idea.

    Design #2

    Second layout: The second layout would be a simplified version with tool tips to show the number of deaths. This allowed for less complexity to the visual.

    Color

    This layout had some elements mixed together to make a simpler version for the eye to follow.

    Layout

    I wanted each graph to show a different story. Therefore, at the top I wanted a visual to show my questions pertaining to age and gender.

    In the lower half, I wanted a visual where it would show the timeline of deaths. However, I thought it would be great if that timeline interacted with the map and only showed the deaths before the date you wanted. This led to me using a feature where the user would hover the mouse and only show deaths before the date hovered over.

    Next, this line graph would be able to show the same visual but split by age and gender to answer the rest of my questions.

    Discovery

  • Does age matter?

    • Younger and older people were affected by deaths. Middle-aged individuals were not as affected.

  • Does gender matter?

    • Chlorea did not affect one gender more than the other.

  • Is there a spike in deaths like a typical outbreak?

    • In general, there was not a direct relationship between age and gender.

  • How does the distribution of deaths change with the date?

    • The outbreak was typical with death numbers from other outbreaks. It follows a normal (ish) shaped curve that starts slow, has a large spike, and dies off slowly.

  • How does the distribution of deaths change with the date?

    • The date does seem to have some correlation with the number of deaths. This could be because the outbreak started slow but spread wuicker as more people got it.
